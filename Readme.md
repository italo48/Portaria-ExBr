# Portaria EB



#### Tecnologias usadas

- Spring Boot
- Thymeleaf
- Hibernate
- MySql
- HTML5 e CSS3
- Bootstrap
- Spring Security

#### Descrição

A Portaria EB é um pequeno sistema de controle de entrada e saida de pessoal.

#### Rodar o projeto

##### Pré-requisitos
- MySql instalado

1. Abra um Terminal
2. Navegue até o diretório raiz do projeto
3. Execute o comando ``mvn spring-boot:run``
